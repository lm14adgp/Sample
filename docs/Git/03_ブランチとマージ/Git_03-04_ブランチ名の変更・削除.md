# ブランチ名の変更・削除

## ブランチ名の変更

コマンド
```
git branch -m [ブランチ名]
```

実行例
```
>git switch feature
Switched to branch 'feature'

>git branch
* feature
  master

>git branch -m new_feature

>git branch
  master
* new_feature

>
```

## ブランチの削除

コマンド
```
# マージされていない変更が残っている場合は削除しない
git branch -d [ブランチ名]

# 強制削除する
git branch -D [ブランチ名]
```

実行例
```
>git branch
* master
  new_feature

>git branch -d new_feature
Deleted branch new_feature (was 9573158).

>git branch
* master

>
```
