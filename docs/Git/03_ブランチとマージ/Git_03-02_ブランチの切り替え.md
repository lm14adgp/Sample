# ブランチの切り替え

## 【旧】ブランチを切り替える

コマンド
```
git checkout [ブランチ名]

# ブランチを新規作成して切り替える
git checkout -b [新ブランチ名]
```

実行例
```
>git branch
  feature
* master

>git checkout feature
Switched to branch 'feature'

>
>git branch
* feature
  master

>
```

## 【新】ブランチを切り替える

コマンド
```
git switch [ブランチ名]

# ブランチを新規作成して切り替える
git switch -c [新ブランチ名]
```

実行例
```
>git branch
* feature
  master

>git switch master
Switched to branch 'master'
Your branch is ahead of 'origin/master' by 7 commits.
  (use "git push" to publish your local commits)

>git branch
  feature
* master

>
```
