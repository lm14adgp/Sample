# ファイルの削除

## ファイル削除

![image of git rm](../../images/Git/01-04_gitrm.png)

コマンド
```
# ファイル/ディレクトリ削除
git rm [ファイル名]
git rm -r [ディレクトリ名]

# ファイルを残したい場合（gitの記録のみ削除する）
git rm --cached [ファイル名]
```
