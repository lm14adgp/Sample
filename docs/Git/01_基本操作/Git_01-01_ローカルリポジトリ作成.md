# ローカルリポジトリ作成

## 自分でプロジェクトを始める場合

コマンド
```
git init
```

実行例
```
>cd C:\Work
>git init
Initialized empty Git repository in C:/Work/.git/

>
```

   > [!NOTE]
   > git initを実行すると、.gitディレクトリが作成される。
