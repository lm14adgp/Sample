# 変更履歴の確認

## Gitのコミット履歴を参照する

コマンド
```
git log

# 一行で表示する
git log --oneline

# ファイルの変更差分を表示する
git log -p [ファイル名]

# 表示するコミット数を制限する
git log -n [コミット数]
```

実行例
```
>git log
commit 030bef8005867aac3f45d25b14c0e17e69717673 (HEAD -> master)
Author: lm14adgp <lm14adgp@gmail.com>
Date:   Sun Dec 28 15:55:50 2025 +0900

    git commit

>git log --oneline
030bef8 (HEAD -> master) git commit

>git log -p index.html

Author: lm14adgp <lm14adgp@gmail.com>

    git commit

>git log -n 1
```
