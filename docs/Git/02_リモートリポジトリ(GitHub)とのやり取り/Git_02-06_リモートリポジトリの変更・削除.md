# リモートリポジトリの変更・削除

## リモートリポジトリ名の変更

コマンド
```
git remote rename [旧リモート名] [新リモート名] 

# 対応するURLを表示する
git remote -v
```

実行例
```
>git remote
origin
test2

>git remote rename test2 test3

>git remote
origin
test3

>
```

## リモートリポジトリの削除

コマンド
```
git remote rm [リモート名] 
```

実行例
```
>git remote
origin
test3

>git remote rm test3

>git remote
origin

>
```
