# GitHubトークンの発行

## GitHubでトークンを発行する

GitHubにサインインし、 **Developer Settings** > **Tokens (classic)** > **Generate new token (classic)**を選択する。

![image of GitHub token1](../../images/Git/02-01_github_token1.png)


 **期間**と**スコープ**を設定し、**Generate token**を選択する。

![image of GitHub token2](../../images/Git/02-01_github_token2.png)
